### Custom Generator Configuration

#### Key 'events'

```
{
  "events": {
    "additional_meta_data_fields": []
  }
}
```

#### Key 'commands'

```
{
  "commands": {
    "additional_meta_data_fields": []
  }
}
```

#### additional_meta_data_fields

Provide additional fields that should be rendererd into meta data in the form ["tenant","correlation_id","causation"]
